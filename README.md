# SQL Bookstore System

## Project Overview

This is a Book Store Management System built using MySQL.

The project uses three tables:

* Books
* Customers
* Orders

## Features

* Manage book inventory
* Track customer information
* Record book orders
* Analyze sales and revenue
* Monitor remaining stock

## SQL Concepts Used

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* JOINS
* Aggregate Functions (SUM, AVG, COUNT, MAX)

## Business Questions Solved

* Total revenue generated
* Most expensive book
* Customers with multiple orders
* Genre-wise sales analysis
* Remaining stock after orders

## Tools Used

* MySQL
* MySQL Workbench
